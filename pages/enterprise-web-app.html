Best Deal
Best Deal is a web enterprise application mimicking the Best Buy website. The website allows customers to browse items by categories, purchase items, view their shopping cart, make a transaction, create accounts, and login and logout. The application uses Apache Tomcat in order to implement Java Servlets. Account login information and transactions are stored in a MySQL database. Customer reviews are stored in MongoDB. The application utilizes object-oriented priciples.
